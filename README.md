# Selen Birincioglu — Portfolio

Personal portfolio website of **Selen Birincioglu** — Industrial Engineering student, digital creator, and creative problem solver.

🌐 **Live:** [selenbirincioglu.netlify.app](https://selenbirincioglu.netlify.app/)

---

## Tech Stack

- **HTML5** — Semantic markup
- **Vanilla CSS** — Custom design system, animations, dark/light theming
- **Vanilla JavaScript** — Interactions, i18n (EN/TR), cursor effects

## Features

- 🌙 Dark / Light mode toggle
- 🌍 Bilingual support (English & Turkish)
- ✨ Custom cursor glow effect
- 📱 Fully responsive layout
- 🎯 Smooth scroll & section animations
- 📬 Contact form
- 🔗 Social media links (YouTube, Instagram, TikTok, LinkedIn)
